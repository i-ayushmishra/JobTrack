
# JobTrack

**JobTrack** is a modern and responsive **job portal web application** where users can search and apply for jobs easily. The project is built using **React.js** and **Tailwind CSS** for a fast and responsive user interface, while **Firebase** and **Local Storage** are used for data management and user authentication.

## 📌 About the Project

JobTrack is designed to simplify the job searching process. Users can **register, log in, search for jobs, and apply for them** based on different filters such as **job role, job type, location, and experience level**.

The platform also displays **recently posted jobs at the top**, making it easier for users to find the latest opportunities. The application uses **Day.js** to manage and format job posting dates efficiently.

## 🚀 Features

* 🔐 **Authentication System** – User **Register, Login, and Logout**
* 🔎 **Job Search** – Search jobs by:

  * Job Role
  * Job Type
  * Location
  * Experience Level
* 📄 **Apply for Jobs** – Users can apply directly through the platform
* 🆕 **Recent Jobs Section** – Latest job postings appear at the top
* 💾 **Data Management** – Jobs can be **added, edited, and deleted**
* 🔥 **Firebase Integration** – Used for storing and managing application data
* 💻 **Fully Responsive Design** – Works on mobile, tablet, and desktop
* 📅 **Date Handling** – Implemented using **Day.js**

## 🛠️ Tech Stack

* **React.js** – Frontend framework
* **Tailwind CSS** – UI styling and responsive design
* **Firebase** – Database and backend services
* **Local Storage** – Temporary client-side storage
* **Day.js** – Date formatting and handling
* **JavaScript (ES6+)**

## 📂 Core Functionalities

1. **User Authentication**

   * Register a new account
   * Login and Logout securely

2. **Job Management**

   * Add new job listings
   * Edit job information
   * Delete job postings

3. **Job Search & Filtering**

   * Filter jobs by role, type, location, and experience
   * View latest jobs first

4. **Job Application**

   * Users can browse and apply for available jobs easily.

## 🎯 Purpose of the Project

The main goal of **JobTrack** is to create a **simple and efficient job portal** that allows users to search and apply for jobs quickly while maintaining a clean and responsive user interface.
